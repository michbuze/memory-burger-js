Règles du Jeu :
Web Burger est un petit JEU DE MÉMOIRE amusant, dont les règles sont simples mais qui est plus difficile qu'il n'y parait.
Des commandes arrivent deux par deux, et vous devez y répondre une par une. Au début arrivent la commande 1 une 2.
Une fois servie la commande 1, les commandes 3 et 4 arrivent et vous devez servir la 2, puis la 5 et 6 et vous servirez la 3...
Peu à peu, tout va s'embrouiller !
Remarque : J'ai imaginé ce jeu lorsque j'étais au collège et je l'avais programmé en Basic sur une calculatrice programmable.
Sources : [ burger.htm ] [ burgert.htm ] [ burgerm.htm] [ rejouer.htm ]
Remarques sur le fichier archive burger.zip contenant l'ensemble des fichiers qui constituent le jeu
- Pour l'utiliser, vous devez extraire tous les fichiers dans un répertoire, plus ouvrir le fichier "burger.htm" avec votre navigateur. Il vous faut un logiciel de décompression zip. Pour Windows, on trouvera sur le site http://izarc.free.fr le logiciel izarc pour le faire (il est gratuit et en français). Vous pouvez aussi consulter ma page d'informations sur les formats d'archive : http://buze.michel.chez.com/123/compress.htm (ou http://z-compress.fr.fm)
- Il n'est pas nécessaire d'être connecté à Internet pour jouer.
- Les fichiers GIF sont les images utilisées par le jeu, et les fichiers burgerm.htm, burgert.htm et rejouer.htm sont utilisés par le fichier "burger.htm", mais ils ne sont prévus pour être ouverts directement par un navigateur.
- Vous pouvez distribuer l'archive à qui vous voulez, sans être obligé de m'en informer, conformément à la licence GPL.
(Par exemple, vous ne devez pas supprimer des fichiers et conserver la licence).
Vous pouvez également inclure ce jeu dans un autre site web, toujours sans être obligé de m'en informer.
- Merci tout de même de me contacter par email à l'adresse michel.buze@gmail.com si vous avez des critiques (bonnes ou mauvaises) à formuler, si vous avez traduit, amélioré, réécrit ce jeu dans un autre langage (ou que vous souhaiteriez le faire).

Remarques sur le code : - Ce jeu a été programmé en Javascript 1.0 pour fonctionner avec une vieille version de Netscape.
- Le coeur de l'algorithme est constitué des fonctions Commande et Teste, soit à peine quelques lignes.
- La fonction animtext sert juste à faire joli... Elle ne marche pas avec Netscape 4.
- Dans le formulaire de burgerm.htm, les éléments de type hidden ne fonctionnent pas avec Netscape 4, ni en indiquant la propriété display:none. J'ai donc utilisé des labels de taille 1px, ce qui est un peu visible avec Firefox.
- J'ai testé ce jeu avec Microsoft Internet Explorer 5.5 et 6.0, Mozilla Firefox 1.0, Opera 6 et une vieille version de Netscape : Communicator 4.7 (ce navigateur est le plus limité), sous Windows XP.
